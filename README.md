# Echo-Guide




ABOUT:

An AI-powered voice-assisted navigation system designed to help visually impaired users understand and navigate their surroundings using speech, camera input, and emergency alerts.


 LIVE FEATURES:

🎤 Voice-controlled assistant (Speech Recognition)

👁️ Live camera access for environment monitoring

🔊 Voice feedback using Speech Synthesis

🔍 Scan surroundings with spoken output

🧭 Navigation guidance system

🚨 Emergency mode with alert system + beep sound

🟥 Emergency UI indicator (red alert mode)



HOW IT WORKS

1. User clicks Start Guide


2. System activates voice assistant


3. User gives voice commands:

“start”

“scan”

“navigate”

“emergency”

4. System responds with:

Voice feedback
Emergency alerts if needed



⚙️ TECH STACK

HTML5 |  CSS  |    JavaScript |    Web Speech API (SpeechRecognition + SpeechSynthesis) |  MediaDevices API (Camera access)

 KEY FEATURES:

🎤 Voice Assistant

👁️ Camera Access

Captures live video feed for potential object detection integration.

🚨 Emergency System

Triggered via voice command or scan logic

Activates red alert UI

Plays beep sound


🧭 Navigation Mode

Guides user step-by-step using voice instructions.

 PURPOSE
